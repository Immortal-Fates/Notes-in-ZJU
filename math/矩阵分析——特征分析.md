# 特征分析

## 特征值和特征向量的定义与求解

定义和求解这里不多赘述

- 特征值的性质
  - 特征值的和等于矩阵的迹：$$\sum_{i=1}^{n} \lambda_i = \text{tr}(A)$$
  - 特征值的积等于矩阵的行列式：$$\prod_{i=1}^{n} \lambda_i = \det(A)$$

- 矩阵的谱

  - 矩阵的谱是指该矩阵的所有特征值的集合：

    $$\sigma(A) = \{\lambda_1, \lambda_2, ..., \lambda_n\}$$

  - 矩阵的谱半径是该矩阵所有特征值的绝对值中的最大值：

    $$\rho(A) = \max \{ |\lambda_1|, |\lambda_2|, ..., |\lambda_n| \}$$

- 矩阵多项式的特征值

  将$A$的特征值代入矩阵多项式即得到多项式的特征值

  > 于是各种奇怪的函数均可以使用泰勒展开再代入矩阵多项式求得
