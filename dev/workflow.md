# 费曼学习学习

1. 找到材料(学习材料=教学材料）
   - 让AI推荐教科书，推荐前置课程的部分内容
2. 写成讲稿(表面目的是让别人听懂，核心目的是让自己听懂)。一份材料收获最多的一定是作者本人，
3. 审核校对(尽可能避免错误，增加一些人性化的细节，帮助理解记忆)
   - 把你的讲稿发给AI，让AI帮你纠错，补充，增加一些比喻（帮助理解和记忆)
4. 复盘总结（总结这次费曼学习法的内容，以及费曼学习法的实践经验）
   - 有什么问题随时问，AI相当于是一个导师，有时候比导师更厉害
   - 你可以以问题的形式加深自己的理解
