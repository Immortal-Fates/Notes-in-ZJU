# Open Class

学习如何学习一些优质的公开课

## How to learn

1. 看官网找课程和其对应的slides

2. 上youtube找到链接，使用[downsub](https://downsub.com/)下载字幕，或者有些课程会提供字幕版

   > [!TIP]
   >
   > Add **subtitle.to/** before the URL and press **Enter**
   >
   > ```
   > subtitle.to/https://www...
   > ```
   >
   > or add **subtitle.to/** between the URL and press **Enter**
   >
   > ```
   > https://www.subtitle.to/youtube.com/...
   > ```

3. slides+subtitle+LLM, begin to learn

4. check the hw and related readings on official website
