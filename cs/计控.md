## 计算机控制系统的设计思想

每一种计算机控制技术的基本特征是什么？

- 功能、安全、成本

  - 可靠性：指在规定的条件下、在规定的时间內系统完成规定的功能的能力

    平均无故障时间MTBF（ Mean Time Between Failure ），可采取的措施：

    采用安全可靠的屏蔽、隔离、接地等抗干扰技术/提高软件可靠性

  - 可维护性：指故障发生后通过维修使系统恢复的能力。本质： 易于查找故障，易于排除故障。平均故障修复时间MTTR (mean time to repair )  提高控制系统可维护性的主要措施：

    - 选择系列化、标准化、通用化、一致性好的硬件设备
    - 设计合理的系统结构
    - 系统最好能带电插拔维修
    - 能够自诊断

  - 实时性：从硬件和软件两方面来考虑，系统又足够快的响应速度

  - 性能价格比：性价比，系统或部件进行选型时，需要综合权衡性能和价格两方面来考虑

## 计算机控制系统的发展历史

计算机控制发展过程：（出现之前是常规仪表控制系统）

- 直接数字量控制DDC(Direct Digital Control)

  本质: 用一台计算机取代一组模拟调节器，构成闭环控制回路，用数字控制技术简单地取代模拟控制技术

  计算灵活，精度高，可分时处理多个控制回路， DDC也很快发展到PID以外的多种复杂控制

  价格贵（当时），功能固化且有限，可靠性低

- 集中型控制系统，控制/危险/信息集中

  用一台计算机来控制尽可能多的回路，回路成本大大降低，信息集中（便于实现复杂控制、优化控制功能）

  由于当时计算机总体性能低，**容易出现负荷过载**，控制集中直接导致**危险集中**，高度集中使系统变得十分“脆弱”

  ![image-20250606220140292](assets/计控.assets/image-20250606220140292.png)

- 集散控制系统DCS，控制/危险分散，信息集中

  控制功能分散到若干个控制站实现，集中测控信息，满足整体管控目标

  **特点：半数字、半分散**

- 分布式IO DCS(Distributed Control System)，模拟信号到IO站点站站间现场总线连接

- 现场总线控制系统FCS，全数字通信全分散控制

  是以现场总线为基础的全数字化的集散控制系统

  **特点：全数字、全分散**

## 计算机控制系统的网络通信技术

### 网络通信基本概念

- 模拟信号（连续的物理量）与数字信号（不连续的物理状态来表示）
- 并行传输（以字/字节为单位同时传输）与串行传输（逐位传输，通信线少，适宜远距离通信，几乎所有计算机控制系统均采用**串行数据通信**）
- 通信双方的交互方式
  - 单工通信：单一方向传输，没有反向交互(如键盘、鼠标通信)

  - 半双工通信：双方可以交互数据，但不在同时，控制简单、可靠、成本低，应用广泛(只需2根通信线)
  - 全双工通信：双方可以在同时交互数据、控制相对复杂，效率高、需2个信道，至少3根通信线(1根为共用地)，或4根

- 基带传输与频带传输

  基带传输：
  - 按数字波形原样、以**“位”流**为基本形式直接在信道上传输
  - 基带网介质便宜（双绞线、同轴电缆、光线等），速度较高，远距离传输容易因信号衰减发生畸变（中继），绝大多数局域网都采用基带传输
  - 数字信号的数字传输

  频带传输：
  - 也称为载波传输，采用**调制、解调**技术
  - 在发送端，将二进制数据，调制变换成一定频带范围的模拟信号；在接收方，通过解调进行反变换。采用模拟信号传送数据时，通常只占用有限的频带宽度。数字信号调制称为“键控”
  - 调制方式主要有：幅度键控(ASK)、频移键控(FSK)、相移键控(PSK)……
  - 数字信号的模拟传输，如HART信号

  宽带传输：
  - 适用于传输影像、语音信息(一般不用于工控网络)

- 计算机网络的拓扑结构

  ![image-20250607120204859](assets/计控.assets/image-20250607120204859.png)

- 网络传输介质

  网络传输介质是指通信网络中数据发送方与接收方之间的物理通路，常用的传输介质有： 双绞线、同轴电缆、 光纤和无线传输等。

  工业控制系统的通信网络多采用： **带屏蔽的双绞线STP、光纤**

- 总线接口技术

  总线（BUS）是指计算机系统中采用的一组公共信号线，它是计算机系统的通信线

  总线的分类：片内总线、系统总线、局部总线、 外部总线（计算机与计算机之间、计算机与仪器/设备之间的连接总线，用以进行组网与通信）

  外部总线类型：串行通信总线、 USB总线、现场总线、工业以太网等

  总线的性能指标：
  - 时钟频率（MHz）
  - 总线宽度（bit位）
  - 总线传输速率（Mbps，每秒兆字节）
  - 同步或异步方式
  - 多路复用
  - 信号线数

### 串行通信总线

串行通信是指构成信息的二进制字符的各位数据一位一位顺序地传送的通信方式

通信技术：

- RS232、RS422、RS485、现场总线基本特点（通信距离、传输速度）

  > RS422、RS485本质上相同，RS485是现场总线“首选”标准

  | 串行通信协议类型 | RS-232   | RS-422 | RS-485 |
  | ---------------- | -------- | ------ | ------ |
  | 接口电路         | 单端     | 差动   | 差动   |
  | 抗干扰能力       | 弱       | 强     | 强     |
  | 标准最大传输距离 | 15m      | 1200m  | 1200m  |
  | 通信双方交互方式 | 全双工   | 全双工 | 半双工 |
  | 标准最大传输速率 | 19200bps | 10Mbps | 10Mbps |

  - RS-232：RS-232C总线标准设有25条信号线， 包括一个主通道和一个辅助通道。 最大工作速率为20Kbps， 最大通信距离为15m
  - RS-485：RS-485是在RS-422基础上制定的标准，采用平衡发送和差分接收，因此具有抑制共模干扰的能力。应用RS-485可以联网构成分布式系统，最大支持**32**个节点，如果使用特制的485芯片，可以达到128个或者256个节点，最大的可以支持到400个节点

### 现场总线通信

现场总线： 连接现场设备、仪表的一种串行、数字式、多点通信的数据总线用于把遵循特定通信协议的控制系统现场仪表、设备连接组网

现场总线技术特征：

- 全数字→现场设备高度自治
- 全分散→系统结构高度分散
- 协议高度开放性（美好愿望而已，不是标准而是标准群）
- 实时以太网技术已被广泛接受

IEC61158 Ed.2-1999  国际标准，本意：排他的和联合的

IEC61158 Ed.3-2003  2003年4月， IEC61158 Ed.3现场总线第三版正式成为国际标准

IEC61158 Ed.4--2007  产生了实时以太网（Real Time Ethernet，简称RTE），亮点： **EPA**

**EPA：我国自动化领域首个国际标准**，意义：打破现场总线核心技术与标准被国外垄断(芯片、专利、标准)的局面

现场总线与一般通信网络的技术特征有何差异？

- 通信速度、可靠性、安全性、网络拓扑

  | 技术特征 | 一般通信网络       | 现场总线                                 |
  | -------- | ------------------ | ---------------------------------------- |
  | 通信速度 | 快（相对）         | 可靠的快<br>最大响应时间是确定的（绝对） |
  | 安全性   | 一般要求           | 本质安全、电磁兼容                       |
  | 网络拓扑 | 网络型<br>(多路径) | 总线型<br>(单一路径且易实现本质安全)     |

  现场总线的本质就是一种通信网络，但二者在基本功能、 信号传输要求和网络结构上有所不同。

  - 基本要求：现场总线，可靠地传递信息，本质安全、电磁兼容

  - 信号传输要求：现场总线通常采用以下二种技术来保证其实时性：简化技术（简化网络结构成线形，简化啊通信模型，简化节点信息几个字节）/网络管理技术（采用主-从、令牌等访问方式，只要限制网络的规模，就可以将响应时间控制在指定的时间内）

    总而言之：**实时性要求（降低不确定性）是现场总线区别一般计算机通信的主要因素**

    > 工控通信的基本特征是什么？
    >
    > - 短帧、周期性→最长完成时间可预知

  - 网络结构：通信网络，网络状的，节点间的通信路径是不固定的

    现场总线，线状的，节点间的通信路径是比较固定的

以太网与工业以太网（如ProfiNet）

工业以太网，以太网技术从办公自动化走向工业自动化

- 早些年工控系统为什么不用Ethernet通信？

  - 介质访问控制协议CSMA/CD→先听后说/边听边说→大负荷时存在碰撞可能→存在“原理上的不确定性”

  - 通信速率↑，网络负荷↓，碰撞概率↓
  - 交换技术发展 → 进一步减轻碰撞问题
  - e网负荷＜10%时，基本无碰撞；负荷＜25%时，e网通信响应时间明显短于令牌网
  - “不确定性”不再是e网用于工控底层网络的主要障碍（工业以太网）

- 工业控制系统网络通信的特点

  - 短帧信息多、长帧信息少
  - 周期性信息多（测量、控制信息），非周期性信息少（操作指令、组态信息；报警等突发性事件信息）
  - 信息流向方向性明显
  - 节点数少、网络负荷较平稳
  - 简化的OSI模型（7层→3、4层）

  工业以太网碰撞概率是很低的。其实以太网的不确定性问题没有那么严重，负荷越小，碰撞越少，且工业控制网络的特点，也利于以太网的应用。“不确定性”不再是E网的主要障碍

思考：远程节点的引入有什么好处、有什么问题、有什么启示？

好处：

- (1)可远程/自动控制设备的工作运行。
- (2)实现了远程监测，对设备参数以及运行状况能够实时了解的到，对各项数据能够及时的分析处理。
- (3)能够对设备进行集中的管理和控制，方便节省时间。
- (4)具有预警、报警功能，及时通知用户或者工作人员，帮助快速发现问题解决故障，防止设备出现更大的问题，增加维修成本的同时可能还会危及人员人身安全。
- (5)工业远程控制系统还可以做到对设备运行工作的情况和数据、故障报警等进行记录，当需要查阅的时候，就能够立马查询的到。
- (6)能够进行远程操作控制，无需人员在现场就能够实现对设备的管理、调试、控制，打破了时间和空间上的限制。
- (7)除了远程控制外，当设备发生故障的时候，还能够远程的进行诊断和维修，而且还有上述的存储记录为数据分析作基础，节省了维修人员在查找故障原因上花费的时间。

## 计算机控制系统的数字滤波技术*

思考题：

有哪些类型的干扰？受到干扰以后怎么办？

模拟信号怎么样处理不容易受到干扰？数字信号怎么样处理不容易收到干扰？

- 干扰类型

  - 脉冲干扰
  - 随机干扰
  - 周期性干扰

- 受到干扰后怎么办？：滤波

- 常用时域数字滤波算法

  - 中位值滤波：连续采样奇数次，将结果从大到小排列取中间值作为采样的有效数据。克服偶然因素干扰，对于温度、液位等缓慢变化参数的滤波效果较好，不太适用于流量、压力等快速变化的参数

  - 算术平均：适用于随机干扰的滤波，对于测量速度慢或响应速度要求较高的系统是无法接受的

  - 递推滤波：对周期性干扰有良好抑制作用，对偶然性脉冲性干扰抑制作用差
  - 加权递推平均滤波
  - 一阶惯性滤波（软件RC滤波）：常好用，取值依据是不产生明显的波纹

  | 滤波器类型           | 适用干扰类型                     | 备注             |
  | -------------------- | -------------------------------- | ---------------- |
  | 中位值滤波法         | 偶然的脉冲干扰                   |                  |
  | 算术平均值滤波法     | 随机干扰                         |                  |
  | 递推平均滤波算法     | 周期性干扰                       | 有良好的抑制作用 |
  | 加权递推平均滤波算法 | 大纯滞后对象和采样周期较短的系统 |                  |
  | 惯性滤波法           | 高频的干扰信号                   | 但不能太大       |

# CH2 检测仪表原理与选型

## 检测仪表概述

### 仪表分类与信号制

仪表类型

- 基地式控制仪表：指示、调节等功能部件集成一体，一台仪表就能解决一个简单控制问题

- 单元组合仪表：按功能分成独立仪表单元，单元间用标准信号联系， 根据需要组合构成控制系统

  - 电动单元组合仪表（DDZ）

    DDZ-Ⅲ型仪表、气动执行器的应用很广泛——集成电路，电源24VDC   信号4～20mA、 1～5VDC

  - 气动单元组合仪表（QDZ）

- 新型仪表： 数字仪表、智能仪表、现场总线仪表

信号制：信号制即信号标准， 用于实现控制系统中不同单元互联互通

- 气动仪表的信号标准：GB777-1985/ IEC382

  | 下限 0% | 上限 100% |
  | ------- | --------- |
  | 20KPa   | 100KPa    |

- 电动仪表的信号标准 GB/T 3369-1989/ IEC381A

  | 序号 | 电流信号 | 负载电阻          |
  | ---- | -------- | ----------------- |
  | 1    | 4-20mADC | 250-750欧         |
  | 2    | 0-10mADC | 0-1000欧/0-3000欧 |

  仪表负载能力是有限的

- 数字仪表的信号标准 RS-232、 RS-485、 **HART**、各种现场总线标准（工业以太网）

- 在布线困难或远程检测控制的场合，**无线通信**，例如： ISA SP100.11a（支持多协议）， WirelessHART（易于无线扩展）， LoRaWAN/NB-IoT（低功耗、远程）

无线通信协议简介

- 无线通信指用无线电波、红外线、光波等作为传输媒体的通信方式
- 常见的无线通信产品及其技术协议
  - 蓝牙，支持设备短距离通信(一般10m内)
  - WiFi，把有线网络转换成无线信号，是当今使用最广的无线网络传输技术
  - ZigBee，一种短距离、低速率、低功耗、灵活组网、高效的无线网络技术，是物联网的主流技术，在工业、农业、智能家居等领域应用广泛

- 两种介质访问控制协议
  - CSMA/CD（ Carrier Sense Multi Access/Collision Detection）： 有线局域网
  - CSMA/CA（ Carrier Sense Multi Access/Collision Avoidance）： 无线局域网

- 上述协议在通信距离、抗干扰性等方面存在局限，常用于消费电子产品，不适合复杂工业现场环境

三种主要的工业无线通信标准

- 国际HART通讯基金会发布的WirelessHART： 是专门为过程工业而设计的开放、可互操作的无线通信标准，只要是基于HART的设备、软件都可方便集成，对大量应用的HART设备而言非常有利，该标准兼容IEEE802.15.4标准
- 国际自动化学会（ISA）发布的ISA 100： 目标是将各种传感器、执行器、无线手持设备等现场自动化设备以无线的方式集成到应用系统中， 可支持集成FF、 HART、 Modbus、 Profibus等多种协议，主要是对IEEE 802.15.4无线通信标准的扩展和优化
- 中国制定的工业无线网络标准WIA-PA： 是由中国工业无线联盟针对工业过程自动化领域制定的工业无线网络标准， WIA-PA也是遵循IEEE 802.15.4体系架构的用于工业过程测量、监视与控制的无线网络系统，可满足工厂自动化无线数据传输需求

家用智能三表（水电气）的无线通信技术：主要采用NB-IoT、 LoRa等无线通信技术， 这两种技术甚至被称为“物联网之王”

- NB-IoT（Narrow Band Internet of Things， 窄带物联网） ：它构建于蜂窝网络之上，需要相对密集的基站部署才能实现广域覆盖
- LoRa（Long Range） ， 是一种超远距离无线传输方案， 使用网关作为媒介（另外布置）。终端功耗更低、 可接入节点数更多，更适合偏远地区数据传输或移动设备跟踪

  ![image-20250607200801471](assets/计控.assets/image-20250607200801471.png)

  直通单座调节阀，直通双座调节阀（阀体内有两个阀芯阀座， 流通能力大），

  三通阀（三个接管口， 分为三通合流阀、 三通分流阀两种，多用于温度调节、 配比调节和旁路调节，其两路输入流体的温差不宜过大）

  蝶阀（注意泄漏量问题，用于食品、 制药等卫生要求较高的场合）

  球阀（转轴带动球体旋转， 具有一定的切断作用，结构简单， 流通能力大，多作为开关阀使用，有O型/V型）

- 流量特性

  - 流量方程和流量系数$K_V$

    选型逻辑：根据设计流量𝑸、介质密度𝝆、工作差压𝜟𝑷，计算调节阀流量系数𝑲，再根据𝑲来选择对应$K_V$（当阀前后压差为100$KPa$，流体密度为1𝒈/𝒄𝒎𝟑，调节阀全开时，每小时通过阀门的流量数(𝒎𝟑数)记为$K_V$表征调节阀的流通能力）的调节阀
    $$
    Q = \frac{K}{10}\sqrt{\frac{\Delta P}{\rho}}, K=10Q\sqrt{\frac{\rho}{\Delta P}}
    $$

  - 流量特性：介质流过调节阀的相对流量与相对位移（即相对开度） 之间的关系
    $$
    \frac{Q}{Q_{max}} = f(\frac{l}{L})
    $$
    理想流量特性 （调节阀前后压差$\Delta P_V$保持不变）

    实际流量特性

    - 串联管道安装：调节阀开度不同，前后压差$\Delta P_V$会变
    - 并联管道安装：因存在固定旁路流量Q2，影响调节阀的调节能力

  - 可调比

    调节阀所能调节的最大流量$Q_{max}$和最小流量$Q_{min}$之比， 反映可调节能力
    $$
    R = \frac{Q_{max}}{Q_{min}} = \frac{K_{max}}{K_{min}}
    $$

    - $  Q_{min}  $ 为调节阀可以调节的最小流量（理想情况下一般为最大流量的2~3%）
    - 低于 $  Q_{min}  $ 所对应的开度，开度变化引起的流量变化很小
    - 最小可调流量不等于泄漏量，泄漏量要比 $  Q_{min}  $ 小很多
    - 受加工限制，R不会太大，一般为30或50

    实际可调比$R_r$，通常实际比理想更小

- 连续（调速）控制

  调速：通过调节电机转速获得所需扭矩

  电机要实现调速控制有哪些常用方法？ **变频**、变压、变阻、变流、变级、步进

  - 直流电机的调速：PWM，只要改变PWM信号的占空比，就可以改变直流电机两端的平均电压，从而实现直流电机的调速。

  - 三相电机的调速—— 变频器

    变频器主要采用交-直-交的变换方式：工频电源 → 整流成直流 → 逆变成𝑓𝑓、 𝑉𝑉可变的交流电源

1. 一般在什么情况下需要考虑采用变频控制？
   - 负荷有富裕需要降低电机速度时——节能降耗（能耗∝转速³，节能效果显著！）
   - 需要调节电机转速时——平滑调速
   - 需要高启动转矩时——提高性能（低速工作时增加输出电压，可改善转矩不足问题）
   - 需要减少大功率电机启动时对电网的冲击——上电缓冲
   - 需要关注过载、过压、欠压保护等功能的场合

2. 有没有不适合变频控制的场景？
   - 需要直接启动的设备（变频器起动需要时间）
   - 高温环境下的应用（变频器散热相对不好）
   - 需要启动大功率负载的设备

  - 模拟量输入模块(A/D、AI)将现场仪表输出的模拟量信号转化为计算机可以处理的数字信号

    - 逐次比较型A/D转换器基本原理

      ![image-20250607235408123](assets/计控.assets/image-20250607235408123.png)

      转换速度很快，有多少位比较多少次，抗干扰能力弱

    - 双积分型A/D转换器基本原理

      - 工作原理：分为两个阶段
        1. $U_i$加入积分器，从0V积分到一个固定时间T，积分器输出$U_0(T) = KU_i T$
        2. 加入负参考电压$U_{ref}$反向积分，记录$U_0$输出$U_0$降回到0V所需要的时间$T_i$
      - 特性：转换速度更慢，抗干扰能力更强

    - $\Sigma-\Delta$型转换器基本原理

      $\Sigma-\Delta$转换， 对环境噪声的抑制能力更强（ 实际上是实现了噪声的整形）

# CH 6 控制系统的工程设计

## 控制系统设计的内容和步骤

控制系统设计的基本原则

- 满足要求：充分考虑工艺要求、工艺性质、操作条件等要素，用控制理论、控制工程的知识、技术和经验，确定系统结构，制定软硬件配置方案和各控制任务的控制方案，**控制结果应满足生产工艺要求**。

- 系统可靠（safety）：在设计控制方案时，首先应确保系统安全可靠，易于维护，投运后出现的任何问题都可能带来损失，甚至是无法估量的损失，细之又细。
- 自主可控（security）：主要指软硬件及核心控制策略的供给。

- 先进有效：使生产过程处在“最佳”状态下运行，力争获取“最大”经济效益。

- 性价比高：控制方案应力求简单、实用、易用，产出/投入比高。

## 控制系统的抗干扰设计

1. 干扰的类型

   干扰（噪声） 是指对系统运行带来不利影响的其他因素

   | 电子噪声 | 固有存在，程度不一                                         | 优化设计           |
   | -------- | ---------------------------------------------------------- | ------------------ |
   | 电磁干扰 | 只要有电场或磁场存在，就会对电子线路产生电磁干扰，很普遍   | 电磁兼容设计       |
   | 机械干扰 | 机械振动或冲击，电气参数参数发生可逆或不可逆的改变         | 防振、避振         |
   | 热干扰   | 温度变化导致器件参数发生变化（温漂），或产生附加的热电势   | 恒温、对称、补偿   |
   | 光干扰   | （特定波长）光线作用激发出电子-空穴对改变载流子密度        | 对光敏元件作光屏蔽 |
   | 湿度干扰 | 改变绝缘体绝缘强度、电介质介电常数等，间接引起电气特性变化 | 浸漆、树脂封灌等   |
   | 化学干扰 | 化学腐蚀；电化学反应形成化学电势                           | 密封、清洁         |
   | 雷电干扰 | 概率不大，后果很严重                                       | 防雷               |

2. 干扰的主要耦合方式

   - 电源线耦合

     电源容易夹带纹波，电源是仪表系统中最主要的干扰源

     ![image-20250606112146388](assets/计控.assets/image-20250606112146388.png)

     解决方法：

     - 选用高品质电源
     - 选用更小内阻的电源
     - 在敏感负载前加装电源滤波器
     - 不同负载尽可能分开供电

   - 公共阻抗耦合

     就是多个电路通过共有阻抗造成的干扰耦合，最容易产生公共阻抗耦合的地方有：公共接地电阻/电感，公共电源内阻

     核心思想：减小公共阻抗，减小电源内阻

     模数混合系统更需要注意公共阻抗问题！  数字系统中的0，1变换在回流地端上会产生高频噪声，因此模数混合系统的数字地与模拟地应作分类布局，最后将数字地与模拟地汇接到电源的地端上

   - 串扰耦合（感性/容性耦合）

     互感，分布电容，减小线-线串扰耦合主要措施：

     - 增加导线间距、减小平行走线长度；

     - 分类走线：强电vs弱点；高频vs低频；模拟vs数字

     - 靠近地面或大型导体构件

   - 电磁场耦合

3. 几种常用的抗干扰技术

   - 外壳（柜体）屏蔽逻辑

     屏蔽的目的
     - 衰减对目标对象的干扰能量

     抑制电场干扰和抑制磁场干扰是有不同的

     - 电场干扰：反射为主，屏蔽体需要接地，电场屏蔽适宜材料为良导体，与屏蔽体厚度关系不大，高电压、小电流干扰源，以电场干扰为主

     - 磁场干扰：吸收为主，理论上可不接地，磁场屏蔽适宜材料为高导磁率材料，与厚度关系很大。低电压、 大电流干扰源， 以磁场干扰为主

   - 电缆屏蔽与接地

     - 屏蔽电缆：用金属丝网、 金属箔层作为屏蔽层包裹芯线， 主要用于信号线传输
     - 铠装电缆：用钢带、 钢丝、 铝带、 铝管等将芯线包起来， 多用于电力线；抗拉、 抗压性好

     绝大多数干扰通过导线传播和辐射的； 低电平信号对外部干扰一般都是很敏感的

     电缆屏蔽的一般性结论：

     - 屏蔽电缆的屏蔽层**不能不接地**；不接地，还不如选用非屏蔽电缆（屏蔽层引入干扰）

- 屏蔽层有三种接地方式：

  - 单端接地：适用于电场、远场干扰的屏蔽；

  - 两端接地：适用于磁场干扰，通过减小感应面积减小磁场耦合电压；两端接地应注意等电位要求

  - 混合接地：可抑制低频和高频电场干扰

 ![image-20250606113909300](assets/计控.assets/image-20250606113909300.png)

 布线小知识

- 布线要尽量短，但要留有裕量（维修改造用，不需要留太多）
- 分类走线：弱电线路和强电线路必须分开；直流和交流、模拟和数字最好分开，至少留点距离

  - 静电防护

     静电来自两种材料的摩擦、 碰撞、 剥离（接触后分离） 的电荷积聚

     随着芯片集成度增高， 其抗静电能力越来越低， 有些芯片被手触摸就可能烧毁，静电感应对**高输入阻抗电路**的影响尤为显著

     静电防护的主要措施

    - 【避免】 系统表面尽可能平滑， 防止静电积聚；避免摩擦起经典， 如化纤类服装；
    - 【疏导】 完全避免不可能， 可在工作台、 地面铺设防静电材料， 穿到防静电服和鞋袜，带防静电腕带；
    - 【中和】 如果疏导还不够， 还可以加装离子风机、 离子风枪来中和静电荷
    - 【其他】 环境保持一定湿度， 可降低静电影响

  - 接地

     接地：将一个点与某个电位基准面（通常是指大地） 用导体连接起来建立低阻导电通道尽管大地为非良导体， 但容量很大， 可把土壤地层理解为等电位面， 作为系统的基准电位。两种类型的接地：

    - 保护地：防雷接地（瞬时高电位、 大电流的快速泄放）保护接地（大功率设备防止电荷大量积累、 绝缘下降等引起的风险）
    - 工作地：信号基准地、 干扰抑制接地

     接地方式：单独接地、 等电位系统、 混合接地系统

     ![image-20250606114922519](assets/计控.assets/image-20250606114922519.png)

     内部连接注意事项

    - 控制系统的工作地和保护地不能在“柜内”混用

    - 应强调“分类汇总”，汇总点离地的接入点愈近与好，且R4越小越好

    - 控制系统的接地电阻通常要求≤4Ω

    - 在各类接地联结中严禁接入开关或熔断器

    - 一个回路中有多个接地点的，应防止直接相连（加隔离装置），避免地电位差的影响

  - 防雷技术

     防雷的本质就是浪涌抑制，在大干扰进入设备前将能量泄放至大地

     ![image-20250606115526173](assets/计控.assets/image-20250606115526173.png)

     电涌(浪涌)保护器SPD有目的限制瞬态过电压和转移浪涌电流，一个浪涌抑制器件内至少需要一个非线性器件

     ![image-20250606115551729](assets/计控.assets/image-20250606115551729.png)

     开关型器件不能安装在火线与零线、火线与地线之间!
