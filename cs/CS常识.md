# Main Takeaway

记录CS一些常识







- “utils” 文件通常用于包含各种辅助函数和工具函数，这些函数在项目中的多个地方可能都会用到
- FPU是CPU中浮点运算单元（Floating Point Unit）的简称，FPU算力的高速稳定发挥，很大程度上决定了电脑的性能，FPU烤机就是对FPU进行一段时间的高负荷的压力测试，并监测CPU的工作状态，以此评估日常工作时CPU工作是否稳定。因此进行FPU压力测试非常有必要
